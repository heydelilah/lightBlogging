# lightBlogging

 一个轻博客。


#### 基本功能：

- 文章 －增删改查
- 评论 －增删改查
- 登陆、注册
- 权限设计（游客、会员、管理员）

#### 涉及的技术：

- Javascript（Jquery)
- HTML, CSS
- Nodejs
- Seajs -前端模块化
- Mongodb －数据库
- [LESS](http://lesscss.org) -CSS预处理语言

#### 引用的插件：

- [Handlebars.js](http://handlebarsjs.com/) -Javascript 模板引擎
- [UIkit](https://github.com/uikit/uikit) －轻量级前端 UI 框架
- [KindEditor](http://kindeditor.org/) －轻量开源 HTML 编辑器

#### 使用的工具：

- Balsamiq Mockups 3 -原型设计
- Koala －LESS编译工具
